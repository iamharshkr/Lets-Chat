# Introduction
This is an anonymous chat app.

# Concepts Used
<ul>
  <li>HTML</>
  <li>JavaScript</>

# Installation
<ul>
  <li>1. npm install</>
  <li>2. node ./app.js</>
</ul>
Then goto your browser and open index.html.
Node Server will run on 8002 port.
